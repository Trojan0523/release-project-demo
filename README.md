# Node.js TypeScript Project

A Node.js project with TypeScript and standard-version for version management.

## Getting Started

### Install dependencies
```bash
npm install
```

### Development
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Release
```bash
npm run release
```

This will automatically:
- Bump version in package.json
- Generate/update CHANGELOG.md
- Create a git commit and tag
